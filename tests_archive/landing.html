🐍 Absorbe a su presa

Aquila es capaz de importar y ejecutar cualquier biblioteca asíncrona y estándar de Python de manera nativa mediante FFI bidireccional.

🌐 Servidor Integrado

Levantar un API REST o endpoint HTTP requiere 3 líneas y 0 librerías adicionales: está construido directamente en el binario sin pesadillas de dependencias.
