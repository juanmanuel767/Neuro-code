# Contribuir a NeuroCode

¡Gracias por tu interés en mejorar NeuroCode! Este es un proyecto abierto para crear herramientas potentes en español.

## ¿Cómo contribuir?

1.  **Reportar Errores**: Abre un *Issue* describiendo el problema y cómo reproducirlo.
2.  **Sugerir Funciones**: Si tienes una idea para mejorar el lenguaje o la librería estándar, estaremos encantados de escucharla.
3.  **Enviar Pull Requests**:
    *   Haz un *fork* del repositorio.
    *   Crea una rama para tu mejora (`git checkout -b feature/nueva-mejora`).
    *   Asegúrate de que las pruebas pasen (`cargo test`).
    *   Envía tu PR con una descripción clara de los cambios.

## Estilo de Código

*   El núcleo del intérprete está escrito en **Rust**.
*   Sigue las convenciones de `fmt` de Rust.
*   Documenta tus cambios en español siempre que sea posible.
